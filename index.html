Configuring Git

  1. Go to VS code and click Terminal. Arrow then new
  2. Check if git is already installed by typing git --version in he command line.
  3. Configure git in your local computer
    • Set your username
      git config --global user.name "your name"
    • sEt your email address
      git config --global user.email "email address"
  4. To initialize the project directory type git init
